## How to Properly Attribute Creative Commons Images

### For CC BY-SA 4.0 / 3.0 / 2.0 licenses:

You must provide:
1. **Title** of the work (if provided)
2. **Author/Creator** name (with link if available)
3. **Source** link to the original
4. **License** name with link to the license deed
5. **Changes** - indicate if modifications were made

### For CC0 / Public Domain:

No attribution is legally required, but it's good practice to credit the creator anyway.

### Example Footer Credit Block:

```html
<footer class="image-credits">
  <h4>Image Credits</h4>
  <ul>
    <li>"DC Street" by Ted Eytan (CC BY-SA 4.0)</li>
    <li>"Arena Stage" by Carol M. Highsmith, Library of Congress</li>
    <li>"Enid A. Haupt Garden" by Another Believer (CC BY-SA 3.0)</li>
    <li>"U Street Metro Station" by Alyo (CC0)</li>
    <li>"Interior of Union Station DC" by Pacamah (CC BY-SA 4.0)</li>
    <li>"Library of Congress" by NealVickers (CC BY-SA 4.0)</li>
    <li>"Washington Dulles International Airport at Dusk" by Joe Ravi (CC BY-SA 3.0)</li>
  </ul>
</footer>
```

### Dedicated Credits Page:

For websites, it's best practice to have a dedicated `/credits` or `/attributions` page that lists all image credits in full, while using shorter attributions near the images themselves.
